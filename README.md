# Claude 启动器

一个简化 Claude Code 启动流程的 Windows 桌面小工具。

## 功能特点

- 📁 图形化文件夹选择
- 💬 支持当前对话/历史对话模式切换
- 💾 自动记住上次配置
- 🚀 一键启动 Claude Code

## 使用方法

1. 运行 `ClaudeLauncher.exe`
2. 选择或输入目标文件夹路径
3. 选择对话模式
4. 点击"启动 Claude"

## 开发

```bash
# 安装依赖
pip install -r requirements.txt

# 运行
python src/main.py

# 打包
build.bat
```
